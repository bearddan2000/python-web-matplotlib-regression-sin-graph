# python-web-matplotlib-regression-sin-graph

## Description
A demo of creating a sin regression line graph.

## Tech stack
- numpy
- matplotlib
- pandas
- scipy

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`
[Availble here](http://localhost)

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
